## 1. Design System Tokens / رموز نظام التصميم

### Spacing scale (use ONLY these)
```
xs:  4px    sm:  8px    md: 16px
lg: 24px    xl: 32px    2xl: 48px    3xl: 64px
```

### Typography (IBM Plex Sans Arabic)
```
H1: 32px / 700 / line-height 1.2
H2: 24px / 700 / 1.3
H3: 18px / 600 / 1.4
Body: 14px / 400 / 1.6
Caption: 12px / 400 / 1.5
```

### Colors (from `lib/core/theme.dart`)
```
Primary (action):    AC.primary       (gold/navy variants)
Background:          AC.bg1, AC.bg2, AC.bg3, AC.bg4
Text:                AC.textPrimary, AC.textSecondary, AC.textDim
Borders:             AC.border
Success:             AC.success (#10b981)
Error:               AC.error (#ef4444)
Warning:             AC.warning (#f59e0b)
Info:                AC.info (#6366f1)
```

### Component sizes
```
Button height:  40px (large), 32px (medium), 24px (small)
Input height:   40px
Card padding:   16px
Modal max-w:    500px (small), 800px (medium), 1200px (large)
Borders radius: 4px (small), 8px (default), 12px (cards), 16px (modal)
```

**Success:** brief flash green, then route to /app.
